Smart Contract Vulnerability Lab

This is where the big money is. Learn to audit high-value DeFi protocols.

Critical Vulnerability: Reentrancy

The DAO hack. An attacker drains funds by repeatedly calling a withdrawal function before the balance is updated.

// VULNERABLE CODE
function withdraw() public {
  msg.sender.call{value: balance[msg.sender]}(\"\");
  balance[msg.sender] = 0; // Too late!
}
The Fix: Checks-Effects-Interactions

Always update state variables BEFORE making external calls.

// SECURE CODE
function withdraw() public {
  uint amt = balance[msg.sender];
  balance[msg.sender] = 0; // Updated first
  msg.sender.call{value: amt}(\"\");
}
